# Experiment Reset And Re-Run Plan

The current experiment archive is useful evidence, but the strategy IDs became
too machine-generated for daily use. The next clean sweep should preserve the old
artifacts for audit and create a new reset-era experiment root with readable
strategy names.

## Reset Principle

Do not overwrite historical experiment folders. Move forward with a new root:

```text
data/experiments_reset_v2/
```

The old root remains historical evidence. The dashboard can point at the new root
after the reset run is complete and validated.

## Naming Scheme

Reset-era strategy IDs should use:

```text
<family>_<behavior>_<number>_<variant>
```

Examples:

```text
regime_pulse_growth_liquidity_01_risk_on_core
regime_pulse_positioning_02_crowding_escape
growth_inflation_rotation_01_growth_disinflation
exposure_alignment_long_only_01_asset_class
inflation_policy_01_fed_lookthrough_guard
```

Each output row should also include `display_name`, which is now generated by
`trade_bot.research.strategy_naming`.

## Re-Run Batches To Test

1. Regime Pulse core:
   - risk-on core when growth, liquidity, and policy are supportive
   - risk-off core when inflation, credit, and liquidity deteriorate
   - neutral hold mode when cycles conflict

2. Growth-inflation rotation:
   - Growth-disinflation: broad equities, growth, momentum
   - Reflation: value, small caps, energy, commodities
   - Inflation: T-bills, commodities, gold, lower duration
   - Deflation: T-bills, duration, quality, minimum equity

3. Positioning and crowding:
   - crowded upside trim
   - washed-out starter re-entry
   - crowded AI-beta escape
   - washed-out broad-market re-risk

4. Long-only Exposure Alignment / Vol-Adjusted Momentum:
   - max long when asset momentum state agrees with macro regime
   - half long when asset momentum state is positive but macro is mixed
   - cash/no position when asset momentum state conflicts with macro regime

5. Fed look-through / inflation guard:
   - stay risk-on while inflation is sticky but liquidity and credit are calm
   - cut risk when inflation acceleration, money/velocity pressure, and Fed
     reaction risk align

6. Integrated operating systems:
   - conservative retirement core
   - balanced macro swing
   - aggressive AI satellite with regime-pulse guardrails

## Validation Requirements

Every reset-era candidate must report:

- full-history metrics
- 30d, 90d, 6m, 1y, 3y, and custom-window metrics
- rolling 1y, 3y, 5y survivability
- walk-forward holdout
- regime-window performance
- turnover and average holding behavior
- allocation history
- drawdown plus allocation chart
- current action and sizing translation
